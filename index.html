My Camp Site

Previous Next

This is the site which helps you to find the different Campgrounds across the world. User can sign up with Username and Password. Authentication is done by PassportJS. User can Add, Update and delete the Campgrounds which is implemented using RESTful routing. User can also comment on perticular Campground. Authorization is implemented so User can only delete or update the posts he created and same concept works for the Comments too. User can see the error and success messages in form of Flash Messages.

Back end development is done by NodeJS.

Database: With the use of MongoDB and MongooseJS, Multiple users can use this Web Application.

Developed by using HTML 5, CSS 3, EJS, ExpressJS, RESTful routing, MongoDB, MongooseJS, Connect-flash, Express-Session, PassportJS
